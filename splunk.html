Splunk.

Capable of collecting the data from a wide range of devices structuring that data and then placing it is database. This can allow real time monitoring of large amounts of systems data to generate alerts. It also provides a quickly searchable record of what happened for event analysis.

Data

A wide variety of machines produce a lot of data from system logs, TCP handshakes and many more, the problem with this data is that each part of it is generally in a different format. Splunk takes this data and gives it some common structure and then places it in a Database that is searchable creating something that can be monitored in real time or examined retrospectively to see what occurred. Once set up these parts need little user interaction.

Searches

The power of Splunk is in its ability to refine down this large data set via the search options.
