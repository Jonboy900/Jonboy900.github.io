Cyber Security Concepts.

Overview - Cyber Security is an ongoing answer to cyber crime which can come in many different forms from Ransomware to stealing sensitive data and just about everything else in betweem. The crimnals in this view could be state sponsored opperations, criminal gangs, political activists or just someone in their bedroom.

Open Systems Interconnection Model - Designed to allow different systems to talk with one another the OSI model is fundamental to understanding many parts of cyber security.

Tools of the Trade

Linux - the opperating system we be using to interface with many of the tools.

  • Linux

Ping - checks an IP adress to see if it is active.

Nmap - will provide a list of ports and services that are running on and active IP.

  • Nmap

Dirb - to look for hidden directories.

Enum4Linux - to identify user accounts.

Hydra - to brute force weak passwords.

  • Hydra

John - to break encription on a file.

Burp Suite - for web application testing.

  • Burp

Metasploit - A framework for penatration testing.

  • Metasploit

Nessus - A vulnerability scanner.

Splunk - An analysis tool for security information and event management (SIEM).
