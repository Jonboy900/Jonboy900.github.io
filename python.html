Python Programming Language

The Python Programing Language.

This will refer to Python 3 version. Python is a simple language with less need to understand memory management than C. Python uses indentation rather than the {} blocks we see in other languages, this tends to makes the documents somewhat full of white space!

Basic Structure - hello world

Variables

Python does not require you to declare variables before using them, or even to declare the variable type.
a = 7 declares the variable a as an interger because it is given an integer with the value 3.
float a = 4.5 declares the variable a as a float as it recieved a float value 4.5. Or you can use a = float(7).
a = "hello" declares the variable a as a string and it recieved the string value hello.
Lists are similar to arrays, alist = [] alist.append(1) will add a value to alist and a = alist[0] will set a to the value in the first possition in the list.

Looping

For loops - int i; for (i = 0; i < 10; i++) {code block} this will set the counter i to 0 test if i has reached 10 and increment i so doing the code block 10 times.

While loops - int n = 0; while (n < 10) {code block, followed by n++;} while this is the same as above we also have some other commands available break; will cause the loop to end and continue; will cause a section to be skipped.

Functions

In C functions are simple, they take a number of inputs and return one value or none. A function must be declared before they are used either in a header file or in the begining of the C file.

Declaration int this_function(int incoming_value) {code block with a return var}. If the function is not ment to return a value the it is declared as a void and no return is required.

Scope, a function is considered local and when it is complete any variables inside it are lost, however you can add Static to the function.

Pointers

Pointers are used to directly point to a part of memory. That part of memory usually holds a variable, we can assign a pointer to a variable by *pointer_to_var = & var; we can then increment var with *pointer_to_var += 1; and what is stored at *pointer_to_var is increased by one.

Structures

Structures contain a set of variables, a structure needs to be declared before it is used.
struct name {
int x;
int y; }
To create a variable of type structure we can set it with struct name coord; To address the variable inside a structure we use the dot so coord.x = 10; We can pass a structure to a function draw(coord).

Typedefs improve the naming of structures
typedef struct {
int x;
int y;
} point;
Now we can decare a new variable of type point by saying point p;

Lastly We can include pointers in a structure allowing strings to be passed.

Function arguments by reference

As Functions make copies of the variables and do not reference the original variable but if we pass the pointer to the variable instead we can alter the original variable.
void addone(int *n) {
(*n)++; }
addone(&n); This passes the pointer to n and will add one to the original variable n in the greater scope.

We can pass a pointer to structure and then alter the original variables with the following:
void move(point * p) {
p->x++; }
