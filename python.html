Python Programming Language

The Python Programing Language.

This will refer to Python 3 version. Python is a simple language with less need to understand memory management than C. Python uses indentation rather than the {} blocks we see in other languages, this tends to makes the documents somewhat full of white space!

Variables

Python does not require you to declare variables before using them, or even to declare the variable type.
a = 3 declares the variable a as an interger because it is given an integer with the value 3.
a = 4.5 declares the variable a as a float as it recieved a float value 4.5. Or you can use a = float(7).
a = "hello" declares the variable a as a string and it recieved the string value hello.
Lists are similar to arrays, alist = [] alist.append(1) will add a value to alist and a = alist[0] will set a to the value in the first possition in the list.

String Formating and Operations

Python has a large number of string formating options
%s to present a string, %d for intergers, %f floats, %.2f for floating point with fixed number of digits. This is encoded as follows: print("Hello %s" % string)

Operators: len(string) returns the length of the string, string.index("a") will return the possition of the first instance of the letter a int the string, string.count("a") will return a count of the number of a's in the string, string.split(" ") will return a list of strings each one taken from the string where the space is the seperator, string.endwith("one") will return TRUE if the string ends with one,

Looping

For loops - for a in numbers:> this will iterate through the list numbers setting a to each one in turn.

While loops - n = 0 while n < 10: code block, followed by n += 1 this will loop 10 times.

break and continue will work on both types of loop break ending the loop and continue skipping the following statements.

else can be used to catch the end of a loop but will not be triggered by a break statement
