analyzing SMB services on a system.

Enum4linux.

Once an SMB service has been detected on a system typically with Nmap. Enum4linux can be deployed to enumerate the service. Enum4linux comes installed on Kali and Parrot linux systems.

Elements

Syntax - enum4linux [options] target ip.

-U - get user list.

-M - show the machine list.

-N - display the namelist dump.

-S - get the sharelist.

-P - show the password policy information.

-G - display the group and member list.

-a - all of the above or full basic enumeration.
