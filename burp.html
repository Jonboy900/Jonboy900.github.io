Burp Suite.

Burp can capture and manipulate the traffic between you and the web server you are looking at. Once captured this traffic can be examined and altered or passed to one of the other tools inside the Brup Suite.

Burp is available at diffent levels, this considers the free or community version which comes with some limitations.

Elements

Proxy - the intercept and modify tool. Capturing and holding requests or responces with a web server.

Repeater - from a capture this can modify a request and resend that request multipul times.

Intruder - this will allow multipul requests to be issued whith changed parameters allowing Brute Force attacks.

Decoder - allows encoding or decoding.

Compare - provides camparrison of data at different levels.

Sequencer - evaluates randomness in things like cookie values.

Extender - for loading external extensions or the inclusion of internal code segments.
