IP - Internet Protocol.

IP is used to communicate across networks.

DNS - Domain Name System

This is used to map names to IP addresses, if you want to visit http://bbc.co.uk the browser must first make a DNS request to reslove the IP address. It is also possible to reverse this search.

IP Address

An IP address usually takes the format 255.255.255.255, this is being extrended with IPv6 but we will focus on IPv4 for now. IP networks are further broken into different sections or subnets by adding a netmask which controls how large the network is. Slash notation /8 gives 16 million IP addresses, /12 about a million, and /16 give about 65 thousand.
