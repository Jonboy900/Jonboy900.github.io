Protocols

.

Telnet - an Aplication layer potocol used to connect to log on and run programs remotely. Telnet listens on port 23 but is no longer considered a secure option as all the data is sent in clear text. Syntax 'telnet machine_ip port'.

HTTP - HyperText Transfer Protocol, used to transfer web pages. The protocol requires an HTTP webserver and an HTTP webbrowser.

FTP - File Transfer Protocol developed to make the transfer of files between different computers and different systems more efficent. FTP servers commonaly listens on port 21. There are two are town modes for FTP Active and Passive.

SMTP - Simple Mail Transfer Protocol one of the protocols for email systems. The steps in sending and recieving and email: 1 an email client (MUA) has a email to be sent and connects to a (MSA) mail Submission Agent. 2 The (MSA) receives the mail cheack it and hands it to the (MTA) Mail Transfer Agent. 3 The MTA transfers the data across the internet to the tragets MTA. 4 The MTA hand the message to the tragets (MDA). 5 The receipet (MUA) collects its Email from the (MDA). SMTP servers listens on port 25 by default.

POP3 - THe Post Office Protocol version 3 is used to download the email messages from the MDA.

IMAP - Internet Message Access Protocol this is an advance on POP3 allowing synchronization of emails across multiple devices.
