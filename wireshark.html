Filtering

Our capture file can be quite large and quickly become far to much to read but by filtering we can limit what we can see at any one time. A simple filter would be to limit the traffic we are looking at to a specific ip address ip.addr==192.192.192.192. This could be refined to say look only at traffic with a specific source and a specific destination address ip.src == 192.192.192.192 and ip.dst == 192.10.10.10

As wll as filtering by Ip we can filter based on protocoltcp.port eq port number or protocal name or udp.port eq port number or protocol name.
